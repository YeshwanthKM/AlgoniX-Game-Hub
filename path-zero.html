Depth First Search (DFS)

Used to generate the maze by exploring paths deeply and creating a unique structure each time.

route

Breadth First Search (BFS)

Finds the shortest path from start to exit by exploring all possible paths level by level.

analytics

A* (A-Star) Algorithm

An optimized pathfinding algorithm that uses distance-based logic to reach the goal faster and more efficiently than basic search methods.

The Innovation

Unlike traditional racing games, Path Zero shifts the focus from mechanical speed to logical efficiency. It provides a visceral comparison between human heuristic navigation and mathematical perfection.
